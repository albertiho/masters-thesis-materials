# All-Competitor Layered Finalists

This run previously included about 4.78 GiB of row-level parquet artifacts under
`splits/` and `scopes/*/splits/`. They are excluded from the GitHub release:

```text
splits/*/injected_rows.parquet
splits/*/predictions.parquet
scopes/*/splits/*/injected_rows.parquet
scopes/*/splits/*/predictions.parquet
```

The omitted files are generated by:

```powershell
python research/training/scripts/run_all_competitors_country_level_layered_finalists.py
```

The compact thesis metrics under `analysis/thesis_metrics/` are generated from
those row-level artifacts by:

```powershell
python research/training/scripts/extract_if_zscore_layered_thesis_metrics.py
python research/training/scripts/analyze_layered_finalists_cross_country.py
```

The committed CSV/JSON/Markdown files are enough to inspect the thesis-facing
tables and figures. Rerun the full script only when the row-level predictions
need to be audited or regenerated. Expect a long run, potentially tens of
hours, depending on hardware.
