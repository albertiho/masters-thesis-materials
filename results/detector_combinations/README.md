# Detector Combination Results

This folder contains layered detector comparison runs. The Git checkout includes
compact metrics, summaries, metadata, and thesis analysis files. The full local
runs also produced large row-level parquet split artifacts, which are omitted
from normal Git storage.

Omitted files are mainly:

```text
*/splits/*/injected_rows.parquet
*/splits/*/predictions.parquet
*/scopes/*/splits/*/injected_rows.parquet
*/scopes/*/splits/*/predictions.parquet
```

Approximate omitted size at cleanup time:

- `all_competitors_all_countries_country_level_layered_finalists`: 4.78 GiB.
- `country1_all_scopes_if_zscore_layered_combinations`: 2.62 GiB.
- `initial_country1_if_zscore_layered_combinations`: 0.97 GiB.

The omitted files store row-level detector predictions and injected-label
metadata. They are generated by the layered detector scripts listed in the
run-specific README files. Rebuilding the full cross-country finalist run can
take tens of hours.
