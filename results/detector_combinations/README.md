# Detector Combination Results

This folder contains layered detector comparison runs. The GitHub release keeps
compact metrics, summaries, metadata, and thesis analysis files, but omits
large row-level parquet split artifacts.

Omitted files are mainly:

```text
*/splits/*/injected_rows.parquet
*/splits/*/predictions.parquet
*/scopes/*/splits/*/injected_rows.parquet
*/scopes/*/splits/*/predictions.parquet
```

Approximate omitted size at cleanup time:

- `all_competitors_all_countries_country_level_layered_finalists`: 4.78 GiB.
- `country1_all_scopes_if_zscore_layered_combinations`: 2.62 GiB.
- `initial_country1_if_zscore_layered_combinations`: 0.97 GiB.

These files are row-level audit artifacts. They are generated by the layered
detector scripts listed in the run-specific README files. Rebuilding the full
cross-country finalist run can take tens of hours.
