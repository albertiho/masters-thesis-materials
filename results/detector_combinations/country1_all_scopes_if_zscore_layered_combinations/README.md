# Country 1 Layered Combination Run

This run evaluates layered IF/Z-score/sanity combinations on COUNTRY_1 scopes.
The full local run produced about 2.62 GiB of row-level parquet artifacts under
`splits/` and `scopes/*/splits/`. Those files are omitted from the normal GitHub
checkout:

```text
splits/*/injected_rows.parquet
splits/*/predictions.parquet
scopes/*/splits/*/injected_rows.parquet
scopes/*/splits/*/predictions.parquet
```

The omitted parquet files are generated by:

```powershell
python research/training/scripts/analyze_if_zscore_layered_combinations.py
```

The included `metrics/`, `analysis/`, `summary.md`, `summary.json`, and
`run_metadata.json` files keep the compact outputs. Regenerating the row-level
files requires rerunning the script and can take many hours.
