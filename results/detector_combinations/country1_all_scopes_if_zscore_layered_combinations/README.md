# Country 1 Layered Combination Run

This run previously included about 2.62 GiB of row-level parquet artifacts under
`splits/` and `scopes/*/splits/`. They are excluded from the GitHub release:

```text
splits/*/injected_rows.parquet
splits/*/predictions.parquet
scopes/*/splits/*/injected_rows.parquet
scopes/*/splits/*/predictions.parquet
```

The omitted files are generated by:

```powershell
python research/training/scripts/analyze_if_zscore_layered_combinations.py
```

The retained `metrics/`, `analysis/`, `summary.md`, `summary.json`, and
`run_metadata.json` files keep the compact outputs. Regenerating the omitted
row-level files requires rerunning the script and can take many hours.
