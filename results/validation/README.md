# Validation Results

This folder may contain row-level validation split artifacts:

```text
*/splits/*/injected_rows.parquet
*/splits/*/predictions.parquet
```

Those files are excluded from the GitHub release by `.gitignore`. Keep compact
metrics, summaries, and analysis files in Git; regenerate row-level artifacts
only when a validation audit requires them.

The main synthetic validation helper is:

```powershell
python research/training/scripts/validate_anomaly_detection.py
```

Smoke validation CSVs for individual forest detectors are generated by:

```powershell
python research/training/scripts/validate_iforest_mh5_smoke.py
python research/training/scripts/validate_eif_mh5_smoke.py
python research/training/scripts/validate_rrcf_mh5_smoke.py
```
