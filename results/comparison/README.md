# Comparison Results

This folder contains detector comparison outputs. Full local comparison runs may
include row-level parquet split artifacts under run directories:

```text
*/splits/*/injected_rows.parquet
*/splits/*/predictions.parquet
```

Those files are omitted from the normal GitHub checkout by `.gitignore`. The
included metrics and summaries are enough for the compact result record.
Regenerate row-level parquet files when an audit needs per-row predictions or
injected-label metadata.

The initial expanded statistical comparison is generated by:

```powershell
python -m src.research.evaluation.initial_zscore_test_run
```

Other comparison runs are produced by scripts such as:

```powershell
python research/training/scripts/compare_detectors.py
python research/training/scripts/compare_granularity_models.py
```
