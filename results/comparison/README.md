# Comparison Results

This folder may contain row-level parquet split artifacts under run directories:

```text
*/splits/*/injected_rows.parquet
*/splits/*/predictions.parquet
```

Those files are excluded from the GitHub release by `.gitignore`. In the current
workspace they are small compared with the thesis-scale tuning outputs, but they
follow the same rule: keep compact metrics and summaries in Git, regenerate
row-level parquet files when an audit needs them.

The initial expanded statistical comparison is generated by:

```powershell
python -m src.research.evaluation.initial_zscore_test_run
```

Other comparison runs are produced by scripts such as:

```powershell
python research/training/scripts/compare_detectors.py
python research/training/scripts/compare_granularity_models.py
```
